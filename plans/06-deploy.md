## Env vars to set on `waystar-hack` Railway service

Pull values from `.env` but **use the internal Postgres URL**, not the public proxy:

```
DATABASE_URL=postgresql://postgres:<pw>@postgres.railway.internal:5432/railway
BETTER_AUTH_SECRET=<copy from local .env>
BETTER_AUTH_URL=https://<railway-generated-domain>
NEXT_PUBLIC_SITE_URL=https://<railway-generated-domain>
STRIPE_SECRET_KEY=<copy from local .env>
STRIPE_WEBHOOK_SECRET=<copy from local .env; may need to be re-issued against the deployed URL>
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=<copy from local .env>
RESEND_API_KEY=<if slice 5 is in>
EMAIL_FROM=<if slice 5 is in>
```

**Two things that trip people up:**
1. `BETTER_AUTH_URL` and `NEXT_PUBLIC_SITE_URL` have to be the **final** Railway domain — chicken-and-egg. Run `generate-domain` first, then set the env vars, then redeploy.
2. `STRIPE_WEBHOOK_SECRET` is specific to the Stripe webhook endpoint. Create a new webhook in Stripe dashboard pointing at `https://<railway-domain>/api/webhooks/stripe`, copy the signing secret from that, set it.

## A11y pass

Target routes: `/pay/<slug>` first (judging weights it at 10%), then `/login`, then `/admin`.

Checklist (from guide §2.4):
- [ ] Labels associated with every form input (`htmlFor` + `id`).
- [ ] Color contrast — run WebAIM contrast checker on any text under 4.5:1; adjust Tailwind colors if needed.
- [ ] Keyboard navigation works end-to-end (Tab, Shift+Tab, Enter, Space, Arrow on selects).
- [ ] Focus indicators visible — most shadcn components have `focus-visible:ring-*` already; verify on custom components in `payment-form.tsx`.
- [ ] `aria-describedby` on inputs with helper text.
- [ ] `role="alert"` on error messages (already in place on login + payment forms).
- [ ] `alt` on all images. There are no raster images on `/pay/<slug>` other than logo — ensure `<img alt={org.name}>` or decorative `alt=""`.
- [ ] Error messages not conveyed by color alone — icons or text prefix like "Error:".

Run:
```
pnpm build && pnpm start
npx lighthouse https://localhost:3000/pay/memorial-health-mar-12 --only-categories=accessibility
```

Fix anything under 90.
